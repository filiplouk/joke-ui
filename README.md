# joke-ui

A simple joke app that fetches random jokes or programming jokes from an API, providing users with a quick laugh with every refresh. Choose between general jokes or tech-related humor for some light-hearted fun!

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
